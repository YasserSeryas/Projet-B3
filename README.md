## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/YasserSeryas/Projet-B3.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Back
   ```

3. Install the dependencies:

   ```bash
   npm install
   npm run dev
   ```

4. Do the same thing about the front

   ```bash
   cd ..
   cd front
   npm install
   npm run start
   ```
